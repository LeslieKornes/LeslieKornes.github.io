MyPet

C# Console Application

The Virtual Pet Project is a C# console application that allows the user to create and interact with a pet. My application allowed the user to choose from 3 different options for a pet, set a name, then feed, water, play with, or clean up after the pet. The tick method randomly increments one of the pets property values, keeping the game going for as long as the user likes.
